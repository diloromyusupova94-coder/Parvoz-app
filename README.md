# ✈️ Parvoz - Arab tili metodikasi loyihasi

Bu loyiha arab tilini shunchaki so'z yodlash emas, balki tilning mantiqiy ildizlari (root system) va ma'no qatlamlarini anglash orqali o'rgatishni maqsad qilgan.

## 🌟 Asosiy g'oya
- **Quruq grammatikadan voz kechish:** Vizual va mantiqiy tahlil.
- **Ildiz tizimi:** So'zlarning "oila"sini ko'rsatish (Masalan: علم -> olim, ma'lumot).
- **Estetika:** Milliy va zamonaviy dizayn uyg'unligi.
